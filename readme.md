# Публичный репозиторий для задания A1 курса "Алгоритмы и структуры данных" НИУ ВШЭ
## Структура репозитория

```
- /data                     - данные (изображения, csv)
    - area.png              - график вычисленных площадей
    - area_err.png          - график вычисленных ошибок
    - areas.csv             - сгенерированные алгоритмом данные
- a1.cpp                    - код для генерации данных
- a1i.cpp                   - файл с решением (для любого входа)
- a1.ipynb                  - ноутбук с визуализацией
- a1.pdf                    - отчет по работе
```
